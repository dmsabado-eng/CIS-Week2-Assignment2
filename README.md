# CIS-Week2-Assignment2

## Summary

For this assignment, I created a C++ program that asks the user for their name, age, height, weight, and test score. The program calculates the user's BMI, gives a BMI category, converts the test score into a letter grade, and prints a personalized message.

## How I Did It

I created a `main.cpp` file and used `cin` to collect user input. I used variables such as `string`, `int`, and `double` to store the information. Then I used if-else statements to determine the BMI category and letter grade. I also used `fixed` and `setprecision(2)` to make the BMI display with two decimal places.
